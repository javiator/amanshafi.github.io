Testing Cookie Banner Integration

This page tests if the cookie banner appears and functions correctly.

Instructions:

  1. Clear your browser cookies for this site
  2. Refresh the page
  3. You should see a cookie banner at the bottom
  4. Open browser console and run debugCookies()

Debug Commands

Open browser console and try these commands:

  • debugCookies() - Check cookie status
  • clearCookieConsent() - Clear all cookies and reload
  • testCustomSettings() - Test custom settings

Expected Behavior

  • Cookie banner should appear at bottom of page
  • Banner should have "Accept All", "Decline", and "Settings" buttons
  • Clicking "Accept All" should hide banner and enable analytics
  • Clicking "Decline" should hide banner and disable analytics
  • Clicking "Settings" should open a modal with cookie options
